# サードスペース概念に基づくAI支援型インタープリテーション・モデルの研究

A Study on AI-Assisted Interpretation Models Based on the Concept of Third Space

## 概要

同志社大学 経済学部 宮崎耕ゼミ 26期 卒業論文の最終発表用Webサイトです。

京都の懐石料理を事例に、AIが異文化間の「第三の空間」（Bhabha, 1994）を共創する媒介者となりうるかを検証した研究の内容をまとめています。

## 公開URL
